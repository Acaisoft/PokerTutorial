# Requirements

* Install NodeJS: [https://nodejs.org](https://nodejs.org)

# Installation

1. `npm install gulp -g`
1. `npm install tsd -g`
1. `git clone https://github.com/Acaisoft/PokerTutorial`
1. `cd PokerTutorial` 
1. `npm install`
1. `tsd install`

# Running

1. `gulp dev:run`
